# 🐍 SLITHER ARENA

**SLITHER ARENA** is a high-performance, visually stunning battle royale inspired by the classic Slither.io. Compete against AI opponents, grow your snake, and dominate the arena with smooth mechanics and premium power-ups!

---

## 🎮 [Play Now!](https://yahya-atif.github.io/SLITHER-ARENA/) 🚀

---

## ✨ Features

*   📱 **Cross-Platform Support**: Seamless gameplay on both Mobile and Desktop.
*   🕹️ **Floating Joystick**: Intuitive touch controls that spawn wherever you touch on mobile.
*   ⚡ **Double-Tap Boost**: Quick-action speed boost for tactical maneuvers.
*   🔥 **Smooth Performance**: Locked high-framerate logic (optimized for 144Hz+).
*   🗺️ **Smart Minimap**: Track your position and nearby enemies in real-time.
*   🌈 **Premium Visuals**: Rainbow trails, screen shakes, and dynamic zoom effects.

---

## 🕹️ Controls

### 📱 Mobile
*   **Touch & Drag Anywhere**: Move your snake using the floating virtual joystick.
*   **Double Tap & Hold**: Activate boost to speed past opponents!

### 💻 Desktop
*   **Mouse Movement**: Steer your snake towards the cursor.
*   **Spacebar**: Hold to activate boost.

---

## 🛠️ Tech Stack

*   🌐 **HTML5**: Structure and Canvas-based rendering.
*   🎨 **CSS3**: Modern UI with glassmorphism and animations.
*   ⚡ **JavaScript (ES6+)**: Custom game engine with delta-time physics.

---

## 🚀 Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/yahya-atif/SLITHER-ARENA.git
    ```
2.  Open `index.html` in any modern web browser.
3.  Enter your name, pick a skin, and start slithering!
